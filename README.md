# Space-Y
Proyecto de ciencia de datos aplicada de certificación IBM Data Science. Este proyecto intenta predecir si la primera etapa del cohete Falcon 9 de SpaceX podrá aterrizar con éxito después del lanzamiento para ser reutilizada en otros lanzamiento y predecir los costos, utilizando SpaceX API.
