# Space-Y
Este proyecto consiste en analizar cómo SpaceX reduce costos reutilizando la primera etapa de su cohete Falcon 9, y usar esa idea para hacer predicciones.

Se asume el papel de científico de datos en una empresa competidora ficticia (Space Y). El objetivo principal es predecir si la primera etapa del cohete aterrizará con éxito y podrá reutilizarse, ya que esto influye directamente en el costo del lanzamiento.

Certificación IBM Data Science
